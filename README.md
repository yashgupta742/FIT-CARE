# 🏋️ FitCare – Smart Health & Fitness Web App

🚀 A modern AI-powered fitness web application that provides BMI analysis, AI-based recommendations, chatbot assistance, exercise videos, and workout tracking — all in one dashboard.

---

## 🔥 Features

- 📊 BMI Calculator with history tracking
- 🧠 AI Fitness Trainer (Workout + Diet suggestions)
- 💬 AI Chatbot for health & fitness queries
- 🎥 Exercise Recommendation System (video-based)
- 📈 Workout Progress Tracking
- 👤 Profile Card with editable user details
- 🏋️ Workout Plans section
- 🌙 Dark / Light Mode toggle
- 📱 Fully responsive design (Mobile + Desktop)
- 🎨 Premium UI with glassmorphism effects

---

## 🧠 Tech Stack

### Frontend
- React.js
- HTML5, CSS3
- JavaScript (ES6+)
- React Router

### Backend
- Node.js
- Express.js
- MySQL Database

### APIs
- AI Integration (OpenAI / fallback logic)
- Custom Exercise API
